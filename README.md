Una tarjeta de débito digital interactiva y visualmente atractiva, diseñada para compartir el número de cuenta de manera fácil y rápida.

## 📝 Personalización

### Cambiar el número de cuenta

En `script.js`, modifica la línea:

```javascript
const ACCOUNT_NUMBER = '638 180 000 048 723 424';
```

### Cambiar la imagen de la tarjeta

1. Reemplaza `Mickey.png` con tu imagen
2. Si cambias el nombre, actualiza en `index.html`:
```html
<img src="tu-nuevo-nombre.png" ...>
<meta property="og:image" content="https://tudominio.com/tu-nuevo-nombre.png">
```

### Cambiar tu número de WhatsApp

En `index.html`, busca y modifica:

```html
<a href="https://wa.me/525531178903?text=...">
    +52 55 3117 8903
</a>
```
